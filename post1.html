- Development Tools -

I know there is some people that wants to know about game development tools, here is a list of what i use to make the games in Gears Connections / personal projects, and recommend to use for game development tools.

Programming

VS Code Used as a IDE for coding.

HaxelFlixel Used as the main game engine based in Haxe.

Unity Used as a alternative game engine for test projects based in C#.

Audio / Music

Fmod StudioUsed as a audio design tool.

LMMS Used as the DAW (Digital Audio Workshop) for composing.

Audacity Used as the Sound editor.

Art / Design / Animations

AsepriteUsed as a pixel art and animations tool.

Tiled Used as the level design editor.

Clip Studio Paint Used as the tool for making the art concept / illustration of the game. I use the Android free ver.

- Warning -

These tools i use some of them is free / paid and i'm already familiar with them, the tools you use don't matter, whats matter is the quality of the game :).
